Gmail for Alfred
==============

*This is work in progress!*

This workflow lets you manage your Gmail mails with Alfred.

## Features

- Background cache refresh
- Uses OAuth 2.0 to authorize the workflow
- Saves your access_token securely in OS X's keychain
